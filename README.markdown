# dotfiles

These are my dotfiles dotfiles for git, vim, bash and tmux.

## About

KISS

## Installing

1. Clone the repo in your home directory.

  `git clone https://github.com/brettchalupa/dotfiles.git`


1. `cd` into the repo.

  `cd dotfiles`

1. Run the install script to symlink the dotfiles.

  `./install.sh`
